Variable Fonts

The main reason for variable fonts is because of the bandwidth costs that web developers and designers must account for. In traditional web fonts, the different styles within a font family (such as regular, bold, and italics) are downloaded in separate font files. This can add up to 500 KB or more of font data, causing latency. Another issue was that different computers had different pre-installed fonts, causing web pages to render in different ways. In these cases, font was limited by the ones that were pre-installed on the computer and styling options for these fonts were restricted to only bold and italic. Thus, variable fonts are used as a solution. Variable fonts allow all the different styles within a family of fonts to be packaged in a single file. The amount of data saved is significant. Variable fonts allow for up to 70% compression. This is done by having a default style, usually regular, and then having five different registered axes to change the style. These five registered axes are the different features of a font: weight, slant, width, optical size and italics. Each axis allows for a plethora of variations between three values: default, minimum, and maximum. For example, the weight axis is used to create the different styles between regular, default, and bold, maximum. This gives designers the choice to implement a vast array of styles, which help richen the reading experience for users. Another example is using the slant axis. The slant axis allows for some fonts to be interpolated from regular to italic. In addition, custom axes can be used to alter additional aspects, such as the size of serifs or the size of the dot in "i." Other custom axes include stroke contrast, vertical alignment, and x-height. Variable fonts allow developers and designers to create web pages that work consistently on different browsers. They no longer have to limit the amount of fonts and typefaces used in order to preserve bandwidth and lower latency. OpenType variable fonts were presented by Apple, Adobe, Google and Microsoft in 2016.

Furthermore, different CSS properties can be used to alter the variable fonts. For example, "font-variation-settings" can be used to take advantage of both the regular axes and the custom axes. Within "font-variation-settings," the syntax for the five main axes are as follows: 'wght' is used for weight, 'slnt' is used for slant, 'wdth' is used for width, 'opsz' is used for optical size and 'ital' is used for italics. Custom axes are any variations a typeface designer comes up with. One common custom axis is grade, done by using 'GRAD.' Grade is the overall density of the typeface, allowing designers to alter the weight without changing the layout of the elements surrounding the text. All in all, variable fonts give web developers and designers the freedom to use different variations fonts to create the best experience possible for users. Without having to worry about bandwidth constraints or being unsure of how a page will render on a particular browser, developers and designers can make the optimal design choices. Variable fonts is a pivotal innovation in the world of web development.
